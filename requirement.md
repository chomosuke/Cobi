# This document ecapsulate the requirement

## User stories

### Account

#### Profile Picture
- As a user, I want to have a profile picture so that I can be identified easily.
- As a user, I want to change my profile picture so that I can always have the profile picture I want.
- As a user, I want to view another user's profile picture in an enlarged form so that I can view it more clearly.

#### Username
- As a user, I want to have a username so that I can be identified.
- As a user, I want to view other user's username so that I can identify them.

#### Password
- As a user, I want to change my password so that I can change a compromized password.

### Message

#### Reply
- As a user, I want to reply to another user's message so that I can keep multiple conversation going.
- As a user, I want to find the message that's being replied to so that I can better understand the context.
- As a user, I want to forward a message to another user so that I can conveniently pass messages along while letting the other user know the authenticity of the message.
- As a user, I want to see who a forwarded message came from easily so that I can know for sure the sender of the message.
- As a user, I want to prevent other user from viewing my conversation with another user so that all conversations remain private.

### Emotions
- As a user, I want to have emojis in my messages so that I can express my emotion.

#### Gif
- As a user, I want to send gifs so that I can express my emotion.
- As a user, I want to search gifs so that I can find what I want to send faster.

#### Sticker Pack
- As a user, I want to send stickers, so that I can express my emotion.
- As a user, I want to add sticker packs to my personal collection so that I can express emotion using the sticker pack I prefer.
- As a user, I want to search for sticker pack so that I can more easily find the sticker pack I prefer.
- As a user, I want to see the most popular sticker pack so that I can more easily find the sticker pack I prefer.

#### Reaction
- As a user, I want to react to messages so that I can express my feelings about other users' messages.
- As a user, I want to react to messages using any emoji I want so that I can express my feelings in the most accurate way.

### Media
- As a user, I want to send photos so that I can share them with other users.
- As a user, I want to send multiple photos as one message so that I can group them together and not make it too hard to scroll through a lot of photos that is sent at once.
- As a user, I want to conveniently view the photos sent so that I can quickly move on with the conversation.
- As a user, I want to send videos so that I can share them with other users.
- As a user, I want to conveniently view the videos sent so that I can quickly move on with the conversation.
- As a user, I want to send files so that I can share them with other users.
- As a user, I want to conveniently view the files sent so that I can quickly move on with the conversation.
- As a user, I want to save any medias sent so that I do anything with them.

### Chat
- As a user, I want to see a list of my chats (i.e. messages with contacts and group chats) sorted my last message time so that I can find the chat I want to message in.
- As a user, I want to search chats by contact name and group chat name so that I can find the chat I watn to message in easily.
- As a user, I want to search for a message in a chat so that I can find past messages quickly.
- As a user, I want to see a list of all images & videos sent in a chat sorted by time sent so that I can find images & videos sent easily.

#### Contact
- As a user, I only want to send messages to a certain group of users (i.e. contacts) so that I can communicate with them and don't get distrubed by spams.
- As a user, I want to search for another user by their username so that I can add them as contact.
- As a user, I want to see a list of my contact so that I can know who are my contacts.
- As a user, I want to delete my contact so that I can stop them from talking to me.
- As a user, I only want users that I approve of to be my contact.
- As a user, I want to prevent some users from finding me so that I don't get distrubed by them seeking my approval to add me as contact.

#### Group Chat
- As a user, I want to create group chats with people in my contact so that I can have a conversation with all of the together.
- As a user, I want to invite other users into a group chat I'm in so that I can have conversation with them and the rest of the group chat.
- As a user, I want to kick member of my group chat out so that I can hide the group chat's coversation from the member.
- As a user, I don't want other users to add me to group chats unless they're in my contact.

### Notification
- As a user, I want to be notified whenever a message is sent by another user in a chat I am in so that I can reply as soon as possible.
- As a user, I want to be notified whenever someone initiate a call in a chat I am in so that I am aware of it.
- As a user, I want to be notified whenever someone react to my message in a chat I am in so that I am aware of it.
- As a user, I want to be notified whenever someone replied to my message in a chat I am in so that I am aware of it.
- As a user, I want to be notified whenever someone mentioned me in a chat I am in so that I am aware of it.

#### Preview
- As a user, I want to preview the message in the notification so that I can quickly decide whether to reply or not.
- As a user, I want to jump to the message in the notification so that I can read it and the context quickly.
- As a user, I want to reply to the messege in the notification without opening it so that I can quickly move on from it.
- As a user, I want to thumbs up to the message in the notification without opening it so that I can quickly move on from it.
- As a user, I want to mark a message in the notification as read without opening it so that I can quickly move on from it.

#### Muliple Devices
- As a user, I don't want to be notified on my other devices when I'm chatting in one device so that I won't be bombarded with duplicated notification.
- As a user, I want to be notified on all my device when I'm not chatting so that I won't miss a notification.

#### Muting
- As a user, I want to mute my chat for some specific amount of time so that I won't be distrubed by it during that time.
- As a user, I want to mute my chat indefinitely so that I won't be distrubed by it.
- As a user, I want to unmute any chat that I previously muted so that I can recieve notification from it again.
- As a user, I want to still recieved notification of calls, reply to my messages, mentions of me and reaction to my messages even if the chat is muted so that I am aware of it.
- As a user, I want to mute all notification for any chat so I won't be distrubed by calls mentions, replies and reactions.

### Call
- As a user, I want to initiate a call within a chat so that I can converse with people via audio.

#### Camera & Audio Toggle
- As a user, I want to turn on or off my camera during a call so that I can show my emotion better.
- As a user, I want to mute during a call so that I can have some privacy or not distrub other users.
- As a user, I want to know if the other user has muted so that I would know that they are not able to respond.
- As a user, I want to deafen during a call so that I won't distrub peoples next to me.
- As a user, I want to be notified when the other user is deafen so that I know they can't hear me.
- As a user, I want to always be muted and have my camera off when I'm deafened so that I won't be heard if I accidentally deafen myself and forget that I'm on a call.
- As a user, I want to check if the other user is deafen so that I know if they can hear me.

#### Call Quality
- As a user, I want to adjust the call volumn infinitely high incase my phone's volumn is maxed out.
- As a user, I want to hear a faster replay of the buffered audio for the other user when the internet cut off so I don't have to ask them to repeat what they said.
- As a user, I want to adjust the speed of the replay so that I can comprehend the replay while making it fast enough that I won't spend too much time on it.
- As a user, I want to adjust the maximum length of the buffered audio so that I don't have to wait for real time chat too long when the internet come back.
- As a user, I want to select the audio quality of a call so that I can choose the best for my internet.
- As a user, I want to choose to auto select audio quality of a call so that I would have the best audio quality for my internet choosen for me.
- As a user, I want to check my internet status so that I can have a peace of mind when the other user is silent.
- As a user, I want to check if any other users is cutted off during a call so that I'd know they might be saying things that I can't hear.
- As a user, I want to check if I am cutted off during a call so that I'd know if they might not hear what I'm saying.
- As a user, I want to check if I'm currently hearing buffered audio from the other user so I'd know what I'm hearing is not what they're currently saying.
